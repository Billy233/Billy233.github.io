嵌入式：DMA转运

一.笔记

c8t6 DMA1 7个通道

类型 起始地址 存储器 用途
ROM 0x0800 0000 Flash主闪存 FLASH_BASE 存储编译后的代码
0x1FFF F000 系统存储器 bootloader / download
0x1FFF F800 选项字节 独立于程序代码的配置（WDG,flash读保护，写保护）
RAM 0x2000 0000 运存SRAM SRAM_BASE ==电脑的内存
0x4000 0000 外设寄存器 PERIPH_BASE 外设配置
0xE000 0000 内核外设寄存器 内核外设配置NVIC Systick

DMA基本结构图：

1.APB1 APB2的外设可以通过 DMA请求 向DMA发出 硬件触发信号（ADC转换完成，串口收到数据时）请求DMA转运数据。

2.CPU或DMA无法写入 Flash，可以读写SRAM。

3.不能在DMA开启时写传输计数器，先用DMA_Cmd disable 再 enable。

4.M2M == 1时软件触发，0时硬件触发。

二.应用DMA数据转运

copy dataA[] 2 dataB[]

三.ADC扫描模式+DMA

ADC扫描模式有数据覆盖的特性ADC_DR寄存器，配合DMA进行数据转运。
